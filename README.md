###### Installation

> [!TIP]
>
> ```sh
> cd ~/.oh-my-zsh/custom/plugins
> git clone https://github.com/realamirhe/ahimi_zsh --depth 1 ahimi
>
> # then add `ahimi` at the end of plugins in .zshrc
> plugins=(... ahimi)
> ```
